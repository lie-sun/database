实验三 HTML 超链接与多媒体应用

【实验目的】

1、掌握超链接的基本语法、属性设置，学会为网页添加各种超链接；

2、掌握书签链接的语法和定义方法，学会使用书签链接设计 Web 页面；

3、掌握在网页中插入并设置图片属性的基本方法；

4、掌握在图片中设置不同热点区域的方法；

5、掌握为网页设置背景音乐的方法；

6、掌握在网页中嵌入音频、视频和 flash 动画的基本方法；

7、掌握滚动文字的设置方法。

【实验内容】

1、为网页添加超链接；

2、制作书签链接；

3、为网页添加图像及图像链接；

4、制作多媒体网页；

【实验步骤】
一、为网页添加超链接

1．新建 HTML 文件；

2．在 body 标记中输入以下代码：

    <h3>外部链接</h3>
    <a href="http://www.baidu.com" title="BaiDu">百度</a><br/>
    <a href="http://www.edu.cn" target="_blank" title="CERNET">中国教育与科研计算机网</a><br/>
    <a href="http://www.sina.com.cn" target="_self" title="Sina">新浪</a>

3．浏览网页，观察超链接的外观效果，说明 target 属性的作用；

4．在 body 标记中继续输入以下代码（注：超链接的路径和文件可以自己指定，要确保存在）：

    <h3>内部链接</h3>
    <a href="sy3/demo.html" target="_black">内部链接</a>是一个指向本网站中的一个页面的链接。

5．说明上一条代码采用的是哪种超链接路径？为什么？

6．在 body 标记中继续输入以下代码（注：超链接的路径和文件可以自己指定，要确保存在）：

    <h3>下载文件链接（使用相对路径和绝对路径）</h3>
    点击<a href="sy3/HTML.chm">下载 HTML 帮助文档</a>

7．在 body 标记中继续输入以下代码（注：email 地址自己指定）：

    <h3>电子邮件链接</h3>
    <a href="mailto:swallowcs@sina.com?subject=hello">给我发邮件</a>
二、制作书签链接

利用书签链接制作如图 3.1 所示的帮助文档。当鼠标单击列表项中的超链接（如：“EditPlus 简介”）时，页面跳转到相应的内容（“EditPlus 简介”）区域；当鼠标单击内容右侧的“返回”时，页面返回到文档开头。

图 3.1 书签链接

1．打开编辑 sy3/sy3-2-a.html 文件；

2．定义书签名称：利用超链接的 name 属性（参照以下代码）分别为“概述、功能、EditPlus 简介、更新历史、语法突出显示”定义好书签名称。（注：书签名称可以自己设定）。

    <a name="bt1">概述</a>
    <a name="bt11">EditPlus 简介</a>

3．制作书签链接：利用超链接的 href 属性（参照以下代码）分别为概述和功能下面的每个列表项以及“返回”设置超链接，链接到相应的书签名称。 

    <a href="#bt11"> EditPlus 简介</a>
    <a href="#bt1">返回</a>

4．浏览网页效果。

三、为网页添加图像及图像链接

1．新建 html 文件；

2．在网页中添加两幅图片，要求：第 1 幅图片的高度为 400、宽度 300，左对齐，替代文本为 t1；第2 幅图片的高度为 200、宽度 300，边框宽度 3，左、右侧空白 50，顶部空白 100。

3．为图片 1 添加超链接，在新窗口中打开“http://www.baidu.com”；

4．在图片 2 中设置一个圆形区域为热区链接，设置映射图片名称为“baidu”，代码如下：

    <map name="baidu">
        <area shape="circle" coords="100,100,80" href="http://baidu.com" alt="百度">
    </map>

5．然后在图片 2 中添加链接代码，代码如下（代码中省略的图片属性保持步骤 2 设置不变）：

    <a><<img src="sy3/t2.jpg" …… usemap="#baidu"></a>

6．浏览网页效果，观察图像链接和热区链接的区别。

四、制作多媒体网页

1．打开编辑 sy3/sy3-4-a.html 文件；

2．在 body 标记中输入以下代码，为网页添加背景音乐：

    <bgsound src="sy3/蔡琴明月几时有.mp3" loop="1">

3．如果要无限次播放，应该如何设置？

4．将步骤 2 代码注释掉；

5．在列表项标记 li 中，用 embed 标记分别插入音频、视频和 flash 动画：

    <li><embed src="sy3/62.swf" loop="true"></embed>
    <li><embed src="sy3/蔡琴明月几时有.mp3" width="300" height="200"></embed>
    <li><embed src="sy3/0303.swf" loop="true" autostart="true" width="300" height="200"></embed>

6．用 bgsound 标记添加的背景音乐和用 embed 标记添加的多媒体文件有什么不同？

7．在 body 的最后一行输入以下代码，添加滚动字幕：

    <marquee behabior="alternate" direction="left" bgcolor="#3399ff" >欢迎来到我的多媒体世界
    </marquee>

8．浏览网页效果，并说明 marquee 标记及其属性的使用方法。

图 3.3 图像画廊

【实验思考】

1、绝对路径和相对路径分别适用于哪种情况的超链接？

2、链接元素（载体）有哪些？链接目标有哪些？

3、图像链接和热区链接有什么不同？

4、背景音乐和嵌入音频有什么不同？

5、说明 img 标记 alt 属性的作用。

6、如何设置滚动的范围和空白空间？
